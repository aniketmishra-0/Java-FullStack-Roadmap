# Contributing to Java-FullStack Roadmap

Thank you for your interest in contributing! 🎉

## How to Contribute

### 1. Fork the Repository
Click the "Fork" button at the top right of this repository.

### 2. Clone Your Fork
```bash
git clone https://github.com/your-username/Java-FullStack-Roadmap.git
cd Java-FullStack-Roadmap
```

### 3. Create a Branch
```bash
git checkout -b feature/your-feature-name
```

### 4. Make Your Changes
- Add new resources, fix typos, or improve the roadmap.
- Make sure links are working and properly formatted.

### 5. Commit Your Changes
```bash
git add .
git commit -m "✨ Add: description of your changes"
```

### 6. Push and Create a Pull Request
```bash
git push origin feature/your-feature-name
```
Then open a Pull Request from your fork to this repository.

## What Can You Contribute?

- 📚 New learning resources (articles, videos, books)
- 🗺️ Roadmap improvements or additions
- 🐛 Fix broken links or typos
- 💡 Suggest new topics or sections
- 🎨 Improve formatting and readability

## Guidelines

- Keep the format consistent with existing content.
- Ensure all links are valid and accessible.
- Write clear commit messages.
- Be respectful and constructive.

## Questions?

Feel free to open an issue if you have any questions or suggestions!
